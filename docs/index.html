DoM - Que Choisir

Presentation page with demo funcion of the game. Maybe try to beat my Personnal Best score of 1860 points ! ;)
This is my first video game that was created in 2015 for a school project. This was just to learn how games are created and this one is not very hard or beautiful.
This project is a game that it consists to go down as long as possible without touch the roof or the floor. The score is growing as long as you are alive. My PB score is 9680 points (around 16 minutes).
PS : This game is not working on mobile devices.
